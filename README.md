# Rofan AI Helper (R+)

로판 AI(`https://rofan.ai`) 전용 Chrome/Edge 확장 프로그램입니다.
**모든 기록과 설정은 내 브라우저(`chrome.storage.local`)에만 저장되며, 개발자나 외부 서버로 전송되지 않습니다.**

현재 버전: `0.31.0`

## 주요 기능

- **목록 편하게 보기** — 카드에 제작자·대화수·사람수·평균 표시, 폴더 색상, 대화방 스티커/메모
- **내 기록 관리** — 오늘 플레이한 캐릭터, 최근 본 캐릭터, 플레이 통계, 팔로우 제작자·캐릭터 정리
- **대화 기록 저장·검색** — 대화 원문을 로컬에 저장하고 방 안에서 검색·바로가기
- **채팅 편의** — `/단축어` 치환, 단어 치환, 고정 인풋
- **응답 재생성 지시사항** — 재생성 시 원하는 방향을 지시(화면·전송엔 안 보이게 처리)
- **유저 로어북** — 나만의 설정/설명을 등록해 대화에 활용
- **AI 이미지 생성** — 대화 장면을 NovelAI / GPT / Gemini / xAI(Grok)로 이미지 생성 (**본인 API 키 사용**, 사용 비용을 원화로 표시)

## 설치 / 갱신

1. Chrome은 `chrome://extensions`, Edge는 `edge://extensions`를 엽니다.
2. 우측 상단 **개발자 모드**를 켭니다.
3. 처음이라면 **`압축해제된 확장 프로그램을 로드`**를 누르고, 압축을 푼 이 폴더를 선택합니다.
4. 이미 설치돼 있다면 확장 카드의 **새로고침(↻)** 버튼을 누릅니다.
5. 열려 있던 `rofan.ai` 탭도 새로고침(F5)합니다.

> 스토어 등록본이 아니라 "개발자 로드"라서 **자동 업데이트가 되지 않습니다.** 새 버전은 다시 받아 폴더를 교체한 뒤 ↻ 를 눌러 주세요.

## 권한 & 개인정보

| 권한 | 용도 |
|---|---|
| `storage` / `unlimitedStorage` | 내 기록·설정을 브라우저에 저장 (외부 전송 없음) |
| `clipboardWrite` | 복사 기능 |
| `downloads` | 생성한 이미지를 다운로드 폴더(`RofanHelper/`)에 저장 |

- 기본 동작은 `rofan.ai` 페이지 안에서만 이뤄지며, 데이터는 **내 브라우저에만** 저장됩니다.
- **AI 이미지 생성 기능을 사용할 때에 한해**, 입력한 프롬프트가 **본인이 등록한 API 키**로 해당 AI 제공사(NovelAI·OpenAI·Google·xAI 등)에 **직접** 전송됩니다. API 키와 설정은 내 브라우저에만 저장되며, **개발자에게는 아무것도 전송되지 않습니다.** AI 기능을 쓰지 않으면 `rofan.ai` 외 어디에도 요청하지 않습니다.
- 외부 요청은 서비스워커의 **허용 목록(정식 AI 제공사 주소)**을 통해서만 나가고, 그 외 주소로는 아무것도 보내지 않습니다.

## 업데이트 요약 (0.24.4 → 0.31.0)

- **태그 가리기** — 등록한 태그가 달린 캐릭터를 모든 목록에서 가림. 자주 보이는 태그 추천에서 클릭 한 번으로 등록 *(0.31)*
- **대화 기록 저장·검색·바로가기** — 대화 원문을 로컬에 저장하고, 대화방 안에서 검색해 바로 이동 *(0.25)*
- **단어 치환** — 지정한 단어를 실시간·전역으로 자동 치환 *(0.26)*
- **응답 재생성 지시사항** — 재생성할 때 원하는 방향을 지시. 화면·전송 메시지엔 노출되지 않게 처리 *(0.27 ~ 0.28, 0.30)*
- **유저 로어북** — 나만의 설정/설명을 등록해 대화에 활용 *(0.29)*
- 다수의 안정화·버그 수정 (대화 검색 정확도, 로어북 노출 방지, 방 간 기록 분리 등)

## 안전성

- 모든 데이터는 로컬(`chrome.storage.local`)에만 저장 — 외부 서버·개발자 전송 없음
- 외부 요청은 정식 AI 제공사/환율 API 주소로만 제한(허용 목록 방식)
- 코드가 공개돼 있어 누구나 직접 확인할 수 있습니다
