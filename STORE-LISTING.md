## 스토어 상세 설명 (Description)

```
Rofan AI Helper (R+) — rofan.ai 이용을 편하게 해주는 비공식 보조 도구입니다.

※ 본 확장은 rofan.ai의 비공식 팬 제작 도구이며, rofan.ai 운영진과 제휴/후원 관계가 없습니다.
※ 모든 기록과 설정은 내 브라우저에만 저장되며, 개발자나 외부 서버로 전송되지 않습니다.

[주요 기능]
• 목록 편하게 보기 — 카드에 제작자·대화수·사람수·평균 표시, 폴더 색상, 대화방 스티커/메모
• 내 기록 관리 — 오늘 플레이한 캐릭터, 최근 본 캐릭터, 플레이 통계, 팔로우 제작자·캐릭터 정리
• 대화 기록 저장·검색 — 대화 원문을 로컬에 저장하고 방 안에서 검색·바로가기
• 채팅 편의 — /단축어 치환, 단어 치환, 고정 인풋
• 응답 재생성 지시사항 — 재생성 시 원하는 방향을 지시(화면·전송엔 노출 안 됨)
• 유저 로어북 — 나만의 설정/설명을 등록해 대화에 활용
• AI 이미지 생성(선택) — 대화 장면을 NovelAI / GPT / Gemini / xAI(Grok)로 이미지 생성.
  본인의 API 키를 사용하며, 사용 비용을 원화로 표시합니다.

[개인정보]
• 데이터는 전부 브라우저 로컬(chrome.storage.local)에만 저장, 외부 전송 없음
• AI 이미지 생성을 쓸 때에 한해, 입력 프롬프트가 본인이 등록한 API 키로 해당 AI 제공사에 직접 전송됩니다(개발자와 무관).
• 자세한 내용: 개인정보처리방침 참고
```

- **카테고리**: 생산성 (Productivity)
- **언어**: 한국어

## 단일 목적 (Single purpose)

```
rofan.ai(로판 AI) 사이트 이용을 보조하는 도구입니다. 사용자의 대화/캐릭터 기록을 로컬에서 정리·표시하고, 채팅 입력 편의(단축어/고정 인풋)와 선택적 AI 이미지 생성 기능을 제공합니다.
```

## 원격 코드 사용 여부 (Remote code)
- **아니요 (No).** 모든 코드는 패키지에 포함되어 있으며, 외부 스크립트를 내려받아 실행하지 않습니다.

---

## 권한 사유 (Permission justifications)

- **storage / unlimitedStorage**
```
사용자의 기록(메모·평점·숨김·플레이 기록·폴더 색상)과 확장 설정을 브라우저 로컬에 저장하기 위해 사용합니다. 데이터양이 많을 수 있어 unlimitedStorage가 필요합니다. 외부로 전송하지 않습니다.
```

- **clipboardWrite**
```
사용자가 요청한 '복사' 기능(내보내기 데이터, 텍스트 복사)에 사용합니다.
```

- **downloads**
```
사용자가 생성한 이미지를 사용자의 다운로드 폴더(RofanHelper/)에 저장하기 위해 사용합니다.
```

- **호스트 권한 — rofan.ai / www.rofan.ai**
```
확장의 핵심 기능(대화 목록·캐릭터 카드 보조 표시, 기록 수집, 채팅 입력 보조)이 rofan.ai 페이지에서 동작하기 위해 필요합니다.
```

- **호스트 권한 — api.novelai.net, image.novelai.net, api.openai.com, generativelanguage.googleapis.com, api.x.ai, api.anthropic.com, api.deepseek.com**
```
사용자가 '이미지 생성' 기능을 켜고 본인의 API 키를 입력했을 때에 한해, 사용자가 선택한 AI 제공사에 이미지/프롬프트 생성 요청을 보내기 위해 사용합니다. 페이지 CORS 때문에 백그라운드 서비스워커에서 허용 목록(이 주소들)으로만 호출합니다. 사용자 데이터를 개발자 서버로 보내지 않습니다.
```

- **호스트 권한 — open.er-api.com**
```
유료 이미지 모델의 사용 비용을 원화로 표시하기 위해 USD/KRW 환율만 조회합니다. 개인정보는 전송하지 않습니다.
```
