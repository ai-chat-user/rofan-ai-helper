# 개인정보처리방침 (Privacy Policy)

**Rofan AI Helper** (이하 "본 확장")의 개인정보 처리방침입니다.
최종 업데이트: 2026-07-06

## 요약

- 본 확장은 **사용자의 데이터를 개발자나 별도의 수집 서버로 전송하지 않습니다.**
- 모든 기록·설정은 **사용자의 브라우저(`chrome.storage.local`)에만** 저장됩니다.
- 분석(애널리틱스)·추적·광고 목적의 데이터 수집이 **없습니다.**

## 1. 수집·저장하는 정보

본 확장은 아래 정보를 **사용자의 로컬 브라우저에만** 저장하며, 외부(개발자 포함)로 전송하지 않습니다.

- 사용자가 만든 기록: 캐릭터/대화방 메모, 평점, 숨김, 플레이 기록, 폴더 색상, 스티커 등
- 확장 설정: 단축어, 고정 인풋, 이미지 생성 옵션 등
- (선택 기능 사용 시) 사용자가 입력한 **AI 서비스 API 키**

이 정보는 브라우저의 로컬 저장소(`chrome.storage.local`)에만 보관되며, 사용자가 확장을 삭제하거나 데이터를 초기화하면 사라집니다. 클라우드 동기화(`storage.sync`)를 사용하지 않습니다.

## 2. 외부로 전송되는 경우 (사용자가 직접 켰을 때만)

**AI 이미지 생성 기능을 사용할 경우에 한해**, 사용자가 입력·설정한 프롬프트가 **사용자 본인이 등록한 API 키**로 사용자가 선택한 아래 AI 제공사에 **직접** 전송됩니다.

- NovelAI (`api.novelai.net`, `image.novelai.net`)
- OpenAI (`api.openai.com`)
- Google Gemini (`generativelanguage.googleapis.com`)
- xAI Grok (`api.x.ai`)
- (그 외 사용자가 설정한 경우) Anthropic (`api.anthropic.com`), DeepSeek (`api.deepseek.com`)
- 비용 표시용 환율 조회 (`open.er-api.com`) — 개인정보 없이 환율만 조회

이 전송은 **사용자와 해당 AI 제공사 사이에서** 이뤄지며, 각 제공사의 개인정보처리방침이 적용됩니다. **본 확장 개발자는 이 데이터를 볼 수도, 받지도 않습니다.** 외부 요청은 확장 내부의 허용 목록(위 주소들)으로 제한되어 그 외 주소로는 어떤 데이터도 전송되지 않습니다. AI 기능을 사용하지 않으면 `rofan.ai` 외 어디에도 요청하지 않습니다.

## 3. 권한 사용 이유

- `storage`, `unlimitedStorage`: 위 기록·설정을 브라우저에 로컬 저장
- `clipboardWrite`: 사용자가 요청한 복사 기능
- `downloads`: 생성한 이미지를 사용자의 다운로드 폴더에 저장
- 사이트 접근(`rofan.ai`): 확장의 핵심 기능 동작
- AI 제공사 도메인 접근: 사용자가 설정한 이미지 생성 기능에서만 사용

## 4. 제3자 제공·판매

본 확장은 사용자 데이터를 제3자에게 **판매하거나 제공하지 않습니다.** 위 2항의 AI 전송은 사용자 본인의 요청·API 키로 이뤄지는 것으로, 개발자를 통한 제공이 아닙니다.

## 5. 데이터 삭제

브라우저의 확장 데이터 초기화 또는 확장 삭제로 모든 로컬 데이터가 제거됩니다. AI 제공사에 전송된 데이터의 처리·삭제는 각 제공사 정책을 따릅니다.
